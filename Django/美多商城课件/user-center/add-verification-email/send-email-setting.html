Django 发送邮件的配置

1. Django 发送邮件流程分析

Django.core.mail 模块给我们提供了 send_mail 函数来发送邮件:

send_mail(subject, message, from_email, recipient_list,html_message=None)

  • subject 邮件标题
  • message 普通邮件正文， 普通字符串
  • from_email 发件人
  • recipient_list 收件人列表
  • html_message 多媒体邮件正文，可以是html字符串

例如：

msg='<a href="http://www.itcast.cn/subject/pythonzly/index.shtml" target="_blank">点击激活</a>'

send_mail('邮件标题', 
          '邮件正文(不带标签)', 
          '发件人', 
          '收件人列表', 
          '邮件正文(可带标签)')

2. 准备发邮件服务器

1.点击进入《设置》界面

2.找到上面的开启服务

3.发送短信, 点击我已发送按钮, 获取授权码

4.牢记授权码

5.完成《授权码》设置

6.配置邮件服务器

在 dev.py 文件中添加如下代码进行设置:

第一个账号配置

# 发送短信的相关设置, 这些设置是当用户没有发送相关字段时, 默认使用的内容:
# 发送短信必须进行的设置:
EMAIL_BACKEND = 'django.core.mail.backends.smtp.EmailBackend'
# 我们使用的 smtp服务器 地址
EMAIL_HOST = 'smtp.163.com'
# 端口号
EMAIL_PORT = 25
# 下面的内容是可变的, 随后台设置的不同而改变:
# 发送邮件的邮箱
EMAIL_HOST_USER = 'itcast888888@163.com'
# 在邮箱中设置的客户端授权密码
EMAIL_HOST_PASSWORD = 'PKOERMBXJFOLXPCB'
# 收件人看到的发件人
EMAIL_FROM = '煤老板<itcast888888@163.com>'

第2个账号配置, 两个选一个即可

EMAIL_BACKEND = 'django.core.mail.backends.smtp.EmailBackend' # 指定邮件后端
EMAIL_HOST = 'smtp.163.com' # 发邮件主机
EMAIL_PORT = 25 # 发邮件端口
EMAIL_HOST_USER = 'hmmeiduo@163.com' # 授权的邮箱
EMAIL_HOST_PASSWORD = 'hmmeiduo123' # 邮箱授权时获得的密码，非注册登录密码
EMAIL_FROM = '美多商城<hmmeiduo@163.com>' # 发件人抬头

说明:

上面的参数中, 前三个不用动.

后面的三个参数换成自己的

其中:

EMAIL_HOST_USER :

​ 自己注册的163的邮箱地址, 作为发送方. 在公司, 即为企业邮箱.

EMAIL_HOST_PASSWORD:

​ 自己设置的授权码.

EMAIL_FROM:

​ 你希望接收者看到的发送者信息.我这里是煤老板公司发送.因此显示这个.
