发送验证邮件接口

重要提示：

  • 发送邮箱验证邮件是耗时的操作，不能阻塞美多商城的响应，所以需要异步发送邮件。
  • 我们继续使用 Celery 实现异步任务。

    1. 定义和调用发送邮件异步任务

1.定义发送邮件任务

在 celery_tasks.email.tasks.py 中添加如下代码: 

# 导入: 
from django.core.mail import send_mail
from django.conf import settings
import logging
logger = logging.getLogger('django')
from celery_tasks.main import celery_app

# 定义一个发送函数, 发送 email:
@celery_app.task(name='send_verify_email')
def send_verify_email(to_email, verify_url):

      # 标题
    subject = "美多商城邮箱验证"
      # 发送内容: 
    html_message = '<p>尊敬的用户您好！</p>' \
                   '<p>感谢您使用美多商城。</p>' \
                   '<p>您的邮箱为：%s 。请点击此链接激活您的邮箱：</p>' \
                   '<p><a href="%s">%s<a></p>' % (to_email, verify_url, verify_url)


      # 进行发送
    result = send_mail(subject, 
               "", 
               settings.EMAIL_FROM, 
               [to_email], 
               html_message=html_message)

    return result

2.注册发邮件的任务：main.py

在发送邮件的异步任务中，我们用到了 Django 的配置文件。

所以我们需要修改 celery 的启动文件 main.py。

在其中指明 celery 可以读取的 Django 配置文件。

最后记得注册新添加的 email 的任务

import os
os.environ.setdefault('DJANGO_SETTINGS_MODULE', 'meiduo_mall.settings.dev')

from celery import Celery

celery_app = Celery('meiduo')

celery_app.config_from_object('celery_tasks.config')

# 自动注册 celery 任务
celery_app.autodiscover_tasks(['celery_tasks.sms', 'celery_tasks.email'])

3.调用发送邮件异步任务

try:
    request.user.email = email
    request.user.save()
except Exception as e:
    logger.error(e)
    return http.JsonResponse({'code': 400, 
                              'errmsg': '添加邮箱失败'})

# 从 celery_tasks 中导入: 
from celery_tasks.email.tasks import send_verify_email
# 调用发送的函数:
verify_url = '邮件验证链接'
send_verify_email.delay(email, verify_url)


return http.JsonResponse({'code': 0, 
                          'errmsg': '添加邮箱成功'})

2. 生成邮箱验证链接

1.定义生成邮箱验证链接方法

注意:

这个函数的主要作用是根据每个人的不同, 生成不同的邮箱验证链接

所以我们要区分每一个人, 怎样才能区分呢?

每个用户对于模型类来说是各个不同的, 所以生成的验证链接也是不同的.

users.utils.py 


# 导入: 
from django.conf import settings

from meiduo_mall.utils.secret import SecretOauth


# 2.生成激活链接
def generate_verify_email_url(request):
    # 1.user_id email
    data_dict = {
        'user_id': request.user.id,
        'email': request.user.email
    }
    # 2.将参数加密
    dumps_data = SecretOauth().dumps(data_dict)

    # 3.拼接 完整的激活路由
    verify_url = settings.EMAIL_VERIFY_URL + dumps_data

    return verify_url

2.配置相关参数

在 dev.py 文件中添加如下代码, 这个是邮箱验证的 域名部分: 

# 邮箱验证链接
EMAIL_VERIFY_URL = 'http://www.meiduo.site:8080/success_verify_email.html?token='

3.使用邮箱验证链接

更改 users.views.py 文件中的 EmailView 类, 把刚刚上面写的字符串替换为下面的函数: 

# 用定义好的函数替换原来的字符串: 
verify_url = generate_verify_email_url(request)
# 发送验证链接: 
send_verify_email.delay(email, verify_url)

4.启动 Celery

启动 worker (消费者) 执行任务.

$ celery -A celery_tasks.main worker -l info
